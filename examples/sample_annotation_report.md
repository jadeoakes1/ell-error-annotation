# Sample Annotation Report (Text Omitted)

This file provides a summary of selected Label Studio annotations from the Error Annotation for English Language Learning project. Original learner texts are omitted because the source corpus is not redistributed in this public portfolio repository.

## What This Demonstrates

- Token-level linguistic error annotation
- Document-level proficiency labeling
- Use of an error taxonomy and label hierarchy
- Quantitative summaries of error density and error type distribution
