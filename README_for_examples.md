# Error Annotation for English Language Learning - Examples

This folder contains examples for the ELL error annotation project. The original learner corpus text is omitted. The files summarize annotation outputs by sample ID, proficiency label, and error category counts.

## Files

- `examples/sample_annotation_report.md`: polished selected examples for readers
- `examples/annotation_summary_no_text.csv`: per-sample annotation counts with no raw text
- `scripts/convert_labelstudio_export.py`: script used to convert a Label Studio export into public summary files
